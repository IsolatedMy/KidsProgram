Django操作MySql详细教程
====
Django中每一个模型model都对应于数据库中的一张表，每个模型中的字段都对应于数据库表的列。<br>
方便的是，django可以自动生成这些create table, alter table, drop table的操作。其次Django为咱们也提供了后台管理模块(Django-Admin),主要功能是通过后台管理来实现客户端的功能，可以对数据进行增删改查。也可以通过该功能进行二次开发。<br>
